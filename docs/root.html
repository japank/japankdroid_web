Cara Root Redmi 4x

Ada dua cara yang bisa sobat lakukan untuk melakukan proses rooting di Redmi 4X yakni dengan menggunakan Magisk dan satunya lagi menggunakan SuperSU.



Saya lebih merekomendasikan untuk menggunakan Magisk karena sobat akan mendapatkan beragam kustomisasi karena beragam module yang ada di dalamnya, menyembunyikan akses root juga lebih mudah jika menggunakan Magisk.

Sedangkan SuperSU hanya bertugas untuk melakukan root pada smartphone sobat dan hanya itu saja. Apapun pilihan sobat tidak ada masalah sama sekali.

1. Download salah satu file dibawah, jika sobat memutuskan menggunakan Magisk maka sobat juga harus mendownload apk Magisk Manager.

Magisk : Magisk Root dan Magisk Manager APK
SuperSU : SuperSU Root

2. Copy file yang didownload ke dalam Redmi 4X, tidak masalah apa itu di memori internal ataupun memori eksternal.

3. Masuk ke mode recovery seperti cara yang sudah sampaikan tadi (tekan dan tahan volume atas dan power).

4. Kali ini beri ijin TWRP untuk memodifikasi sistem dengan cara swipe layar.

5. Pilih “Install” dan pilih file Magisk atau SuperSU yang sebelumnya sudah sobat copy tadi.

6. Setelah itu, swipe layar dibagian bawah untuk memulai proses flashing.

7. Setelah proses flash selesai, sobat akan diarahkan ke opsi “Reboot System”, pilih opsi tersebut untuk merestart Xiaomi Redmi 4X sobat.

8. Jika sobat tadi memilih memakai SuperSU, maka cara ini sudah berakhir namun jika sobat memilih menggunakan Magisk, maka instal terlebih dahulu Magisk Manager yang sudah sobat download tadi.

Untuk membuat sobat lebih yakin lagi, sobat dapat mendownload aplikasi Root Checker di Play Store untuk kembali mengecek apakah Redmi 4X milik sobat sudah berhasil di root.
