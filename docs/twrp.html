Cara Install TWRP

Karena Redmi 4X ini sudah mendukung untuk dapat di buka bootloadernya, akhirnya smartphone ini pun sudah mendapatkan versi TWRP sendiri. Dan TWRP disini adalah versi TWRP yang resmi. TWRP sendiri merupakan custom recovery untuk Android yang dapat digunakan untuk install/flash ROM, boot image dan lain – lainnya ke dalam smartphone sobat, secara sederhananya sobat dapat melakukan modifikasi apapun ke dalam sistem operasi Android.

Sobat juga dapat melakukan proses root lewat TWRP ini, jadi jika smartphone milik sobat sudah mendukung dan diinstal TWRP atau custom recovery lainnya, maka smartphone sobat hampir dipastikan dapat di root. Namun untuk dapat menginstal/flash TWRP, sobat harus membuka bootloader Redmi 4X milik sobat dahulu.

Pertama, sobat mesti mensetting ADB di komputer terlebih dahulu, caranya yakni download ADB disini.

Kemudian double klik atau jalankan filenya.

Nantinya sobat akan melihat jendela command prompt dengan background biru. Ketik Y dan enter terus menerus hingga kurang lebih tiga kali.

Setelah ADB selesai terinstal, maka jendela cmd itu nantinya akan tertutup sendiri dan sobat sudah sukses menginstal ADB

Setelah bootloader sudah berhasil terbuka, download TWRP untuk Redmi 4X disini kemudian ikuti langkah – langkah dibawah ini.


1. Buka folder tempat hasil download file TWRP tadi.

2. Kemudian di addres barnya, ketik cmd dan tekan Enter. Jendela command prompt akan muncul.

3. Sekarang reboot Redmi 4X milik sobat ke mode fastboot, caranya yakni pertama matikan Redmi 4X sobat lalu tekan dan tahan tombol volume kebawah dan power selama kurang lebih 7 detik, nanti akan muncul logo fastboot.

Namun jika hal itu tidak berhasil, maka coba pakai tombol volume bawah + home + power secara bersamaan.

4. Jika sudah masuk mode fastboot, maka colokkan Redmi 4X sobat ke PC menggunakan kabel USB, kemudian cek layar smartphone sobat, jika ada perintah untuk “Allow USB Debugging”, pilih Yes atau OK.

5. Selanjutnya sobat install TWRPnya dengan cara ketik perintah ini di jendela command prompt

fastboot flash recovery twrp-3.1.1-0-santoni.img

6. Kemudian restart Redmi 4X sobat dengan perintah ini.


fastboot reboot

Itulah cara menginstal atau flash TWRP di Xiaomi Redmi 4X, untuk mengeceknya sobat dapat menulis perintah berikut di cmd tentunya dengan keadaan smartphone yang masih menancap ke PC. Namun saya sarankan untuk TIDAK membuka TWRP pada saat ini.

adb reboot recovery

Namun jika sudah sobat cabut, maka sobat dapat masuk ke mode recovery dengan cara matikan smartphone terlebih dahulu lalu tekan dan tahan tombol volume keatas dan power (jika tidak berhasil maka gunakan tombol volume keatas + home + power).

Ketika sobat pertama kali berada di mode recovery TWRP, maka akan ada TWRP akan menanyakan ijin untuk memodifikasi sistem. Namun sobat jangan pernah memberi ijin hal itu, pilih saja “Keep Read Only”.

Sebenarnya sobat dapat saja mengijInkan untuk mengotak – atik sistem ketika sobat sudah mengcopy file SuperSU atau DM verify disable.

Namun, saya perkirakan sobat belum kan ?

Oleh karena itu, untuk saat ini, sobat jangan ijinkan TWRP untuk mengotak – atik sistem terlebih dahulu. Alangkah lebih baik lagi jika sobat TIDAK perlu membuka TWRP ini sebelum melakukan proses root
